Password Generator

This app is designed to generate a password from randomised characters. A series of arrays are used to select Upper, Lower, Numeral and Special case characters which it presents in randomised order. See the ReadMe for more details.

Work Scheduler

This app was designed to help the user schedule a busy day. Hourly time slots allow the user to enter tasks or appointments, while tracking the time of day. Time slots in the past are green, those in the past are grey and the current hour is red. Practical and simple. See the ReadMe for more details.

Weather Dashboard

This app is designed to allow a user to check the weather in any city in the world, whether its a potential destination or a place remembered fondly.

It provides the user with various data, from temperature to windspeed and humidity.
